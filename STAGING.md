# Staging — شهر تجريبي

فرع `staging` للنشر على ريلواي والاختبار لمدة شهر قبل الإنتاج.

## الفروع

| الفرع | الاستخدام |
|-------|-----------|
| `staging` | ريلواي — اختبار المستخدم |
| `main` | الإنتاج لاحقاً (بعد الموافقة النهائية) |

## ريلواي — الواجهة

1. مشروع ريلواي → خدمة `aquawash-frontend`
2. **Settings → Source → Branch:** `staging`
3. **Build Variables** (مهم — وقت البناء):

```
NEXT_PUBLIC_API_URL=https://<رابط-الخادم>.up.railway.app/api
NEXT_PUBLIC_USE_API=true
```

4. بعد النشر: افتح الرابط وجرب تسجيل الدخول

## روابط الدخول

- لوحة الإدارة: `https://<رابط-الواجهة>/en/admin/login`
- نقطة البيع المحمولة: `https://<رابط-الواجهة>/en/holder/pos`

## بعد شهر الاختبار

1. دمج `staging` → `main`
2. إعادة البناء على `main` مع متغيرات الإنتاج
3. تحديث `NEXT_PUBLIC_API_URL` لرابط الخادم الإنتاجي
